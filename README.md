# Progetto-MCF
Progetto per l'esame di metodi computazionali per la fisica.\
Tutte le analisi sono eseguite sia per i dati raccolti su base mensile che settimanale.\
Lo script python analizza le curve di luce delle quattro sorgenti.\

Per poter funzionare lo script ha bisogno che dal punto del FileSystem in cui è eseguito, la cartella "sorgenti" sia esattamente un livello sotto, quindi accessibile tramite 'cd sorgenti/'.\
Per farlo funzionare altrimenti serve modificare lo script inserendo il percorso relativo corretto per la cartella "sorgenti" nel punto in cui si creano le istanze della classe.\

Il programma può mandare a schermo vari risultati sotto forma di grafici o DataFrame.\
La gestione di queste funzionalità avviene tramite riga di comando da terminale.\
Tutte le funzionalità possono essere visualizzate tramite comando  "--help" o "-h".\
